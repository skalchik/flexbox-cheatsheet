A Guide to the Basics of Flexbox

Term Function When declared
Display Iitializes element to act as either a flex (block level, child elements contained) or inline flex (items inline) element Container
Justify-Content Specifies behavior of child items along the container's main axis Container
Flex-grow Grow to fill a container, numbers inputed are proportional to each other. Default is 0 Item
Flex-shrink Shrink to fit a container, numbers inputed are proportional to each other. Default is 1 Item
Flex-basis Set intial height or width of flex grow or shrink element Item
Flex Shorthand for flex grow, shrink, and basis Item
Flex-wrap Overflow items move to a new line Container
Align-content Spaces rows in containers on the cross axis Container
Align-items Space items on a single row in containers on the cross axis Container
Flex-direction Switches main and cross axis Container
Flex-flow Shorthand for flex-direction and flex-wrap Container
